== Description ==
Gravity forms sells addons will connect your gravity form to your Sellsy account

== Installation ==
* 1 - unzip
* 2 - upload to wp-content/plugin
* 3 - Go to your dashboard to activate it
* 4 - have fun!

== Frequently Asked Questions ==
= Puis je lier n'importe quel formulaire? =
Non! vous devez utiliser (pour le moment) le formulaire fourni (disponible dans l'onglet Options).

= Faut il régler quelque chose sur mon compte Sellsy? =
Oui, vous devez créer une application Sellesy pour obtenir les Tokens nécessaire. Une fois connecté av otre compte Sellsy, rendez vous sur https://www.sellsy.com/?_f=prefsApi
